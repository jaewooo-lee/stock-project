카카오톡 연동 설정

본 프로그램은 카카오톡 REST API를 사용하여 나에게 보내기 기능으로 보고서를 전송합니다. 최초 연동을 위해 인증 절차가 필요합니다.

  1. .env 파일에 카카오 디벨로퍼스 앱의 REST_API_KEY를 등록합니다.
  2. 아래 [카카오 인증 시작] 버튼을 눌러 카카오 계정 로그인을 진행합니다.
  3. 인증이 성공하면 브라우저 화면에 출력되는 Refresh Token을 복사하여 .env 파일의 KAKAO_REFRESH_TOKEN 값에 입력하고 컨테이너를 재시작합니다.
인증 Redirect URL: http://localhost:8000/api/kakao/callback
